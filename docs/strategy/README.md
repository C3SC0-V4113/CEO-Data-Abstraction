# Strategy

Esta carpeta contiene opciones y lineas de estrategia para resolver el sistema
Text-to-SQL asistido por IA con web SSR y MCP remoto.

## Estrategia Vigente

La estrategia principal es [guided-analytics-experience.md](guided-analytics-experience.md):
una experiencia chat-first para CEO, con solo dos interfaces en el MVP:

- login;
- chatbot ejecutivo con reportes generados bajo demanda dentro del chat.

## Documentos

- [guided-analytics-experience.md](guided-analytics-experience.md): estrategia

- [solution-options.md](solution-options.md): comparacion de alternativas.
- [mcp-first-access.md](mcp-first-access.md): servidor MCP remoto y herramientas
  controladas sobre el mismo pipeline Fastify.
- [automated-insights.md](automated-insights.md): reportes, jobs y alertas como
  hipotesis de brainstorming.
- [dashboard-reporting.md](dashboard-reporting.md): opcion historica
  report-first, conservada como referencia y no como estrategia vigente.
