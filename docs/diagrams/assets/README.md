# Diagram Assets

No se agregan logos externos en el MVP de diagramas.

Si se agregan imagenes despues:

- Usar logos oficiales o assets con licencia clara.
- Preferir SVG/PNG livianos.
- Documentar fuente y licencia.
- Evitar que los diagramas dependan de imagenes remotas para entenderse.

Los diagramas actuales usan nombres de tecnologias y agrupaciones visuales en
lugar de logos.
